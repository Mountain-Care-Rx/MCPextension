MCP Extension Updates

This site hosts the latest update files for the MCP browser extension (Chrome, Edge, Firefox).

Update files directory:
/updates/
  • dist-chrome.crx (Chrome extension package)
  • dist-edge.crx (Edge extension package)
  • dist-firefox.zip (Firefox extension zip)
  • .xpi (Firefox signed package)
  • chrome-updates.xml (Chrome update manifest)
  • edge-updates.xml (Edge update manifest)
  • firefox-updates.json (Firefox update manifest)
  • updates.xml (Multi-browser update manifest)
  • dist-chrome.zip (Chrome build zip)
  • dist-edge.zip (Edge build zip)

For more information, see the ChatServerUpgrades.md file or contact the maintainer.
